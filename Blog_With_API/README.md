# Blog with API - MSD OCT22

Project created by [Antonio Rufino](https://github.com/devs-toni) & [David Pizarro](https://github.com/DTPF)

## Requriments

- [node.js]

## Install

```bash
git clone
cd Blog_With_API
npm install
```

## Start

```bash
cd Blog_With_API
npm run dev
```

## Run local database

```bash
cd Blog_With_API
npm run db
```

[node.js]: <http://nodejs.org>
